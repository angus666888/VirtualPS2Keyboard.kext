# Virtual PS2 Keyboard for macOS

这是一个基于 macOS 内核驱动实现的虚拟 PS2 键盘，配套 SwiftUI 编写的悬浮 UI 界面。

## ✨ 已实现功能 (v1.0)
- **内核驱动模拟**: 基于 IOKit 实现高拟真 PS2 键盘按键模拟。
- **不抢占焦点**: 采用 `Non-activating Panel` 技术，点击虚拟键时不夺取当前输入框的焦点。
- **始终置顶**: 悬浮窗始终保持在所有窗口最前端。
- **组合键支持**: 自动处理 Shift 等修饰键状态。
